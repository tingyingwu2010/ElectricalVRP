Vehicle fleet
Type (k) = type of EV 
MaxCap (Q^k) = maximum load capacity of vehicle type k in kg
MaxBat (Y^k) = maximum battery capacity of vehicle type k in KWh
ChargeRate (g^k) = charging rate of the EV type k in kWh/km
Con (e^k) = energy consumption of the EV type k in h/kWh
